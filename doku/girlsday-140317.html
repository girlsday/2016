Internet und World Wide Web

  • Das Internet sind miteinander verbundene Rechner
  • Das World Wide Web sind miteinander verbundene Dokumente
  • Beides wird jetzt synonym verwendet

Was passiert im Netz?

  • Der Browser ruft eine Adresse auf
  • Von dieser Adresse bekommt er den Seiteninhalt
  • Der Browser rendert die Seite und zeigt sie mir an

HTML -
die Sprache des Internets

  • Alle Seiten im Internet sind in HTML geschrieben.
  • HTML ist eine Auszeichnungssprache.
  • Ein Text besteht aus verschiedenen Bestandteilen wie z.B.
    • Titel
    • Untertitel
    • Paragraph

Das Grundgerüst

<html>
  <head>
    <title> Das HTML-Dateigerüst</title>
  </head>
  <body>
     Und hier kommt mein schöner Text mit Überschriften, Links und Bildern hin
  </body>
</html>

Wichtige Text-Tags

  • Überschriften: <h1>, <h2> und <h3>
  • Paragraph: <div>
  • Bild: <img>
  • Link: <a>

Tags - unter der Lupe

  • Es gibt einzelne oder paarweise Tags:
    • Einzeln, z.B.
      Zeilenumbruch <br />
    • Paarweise, z.B.
      Überschrift <h1>Ich bin eine Überschrift</h1>

Tags - unter der Lupe

  • Manche Tags haben Parameter, z.B.
  • <a href="http://youtube.com">Ich bin ein Link</a>
  • Sieht dann so aus: Ich bin ein Link
